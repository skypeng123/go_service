# go_service

#### 项目介绍

服务中心

#### 软件架构

golang+gin


#### 安装教程

```
go get -u github.com/kardianos/govendor

cd $GOPATH/src
git clone https://github.com/skypeng123/go_service.git
cd go_service
govendor sync
go run main.go
```


#### 使用说明

编译:
```
go build -tags=jsoniter .
```
